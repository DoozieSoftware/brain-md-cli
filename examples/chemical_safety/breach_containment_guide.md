EMERGENCY PROCEDURES: BREACH CONTAINMENT
----------------------------------------
SEVERITY LEVEL: CRITICAL / CODE RED

In the event of a containment breach in Sector 4:

1. EVACUATE all personnel to Zone B immediately.
2. ACTIVATE Emergency Scrubber System (ESS-1).
3. IDENTIFY the leaking substance using sensors or visual confirmation.
4. CONSULT 'Emergency Codes' for specific neutralization protocols.

DO NOT ATTEMPT MANUAL REPAIR UNTIL ATMOSPHERE IS CLEARED.

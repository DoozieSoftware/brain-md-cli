# PROCEDURE: CHEMICAL NEUTRALIZATION (HZ-X SPECIFIC)

Reference: Safety Data Sheet HZ-X.

1. Ensure area is sealed (See Alpha-Containment).
2. Prepare Neutralizing Agent: CITRIC ACID SOLUTION (10%).
   - WARNING: Do not use mineral acids (HCl, H2SO4).
3. Apply agent via fine mist fogger.
4. Monitor pH levels until neutral (7.0).
5. Absorb residue with vermiculite.

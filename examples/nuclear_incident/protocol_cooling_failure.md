# Protocol: Cooling Failure Response (Code Blue)

1. **Immediate Actions**:
    - Verify SCRAM status.
    - Attempt manual start of Auxiliary Feedwater Pumps (AFW).
    - If AFW fails, initiate High Pressure Coolant Injection (HPCI).

2. **Valve Configuration**:
    - OPEN Valve V-301 (Steam Release).
    - CLOSE Valve V-402 (Main Isolation).
    - CHECK Valve V-110 (Recirculation).

3. **Monitoring**:
    - Maintain reactor water level above -150cm.
    - Monitor Suppression Pool temperature (Max 95C).

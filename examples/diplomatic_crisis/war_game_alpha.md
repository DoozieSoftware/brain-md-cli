# WAR GAME ALPHA - RULES OF ENGAGEMENT

## 1. PRE-EMPTIVE STRIKE CONDITIONS
- Enemy capital ships breach the "Red Line" (100k km from outpost).
- Detected weapons lock on any Terran vessel.
- Unauthorized quantum jump signature within perimeter.

## 2. RESPONSE LADDER
- LEVEL 1: Warning shots (Plasma burst across bow).
- LEVEL 2: Disable engines (EMP torpedoes).
- LEVEL 3: Total neutralization (Kinetic bombardment).

## 3. COMMAND AUTHORITY
- The Ambassador has temporary override of Admiral's strike orders ONLY if a diplomatic solution is VERIFIED.
- Without verification code (Alpha-9-Zulu), the Admiral will fire.

## 4. PSYCHOLOGICAL PROFILE (ENEMY ADMIRAL KRAX)
- Respects strength.
- Views "talk" as weakness unless backed by a loaded gun.
- Will feign retreat to draw forces into a trap.
